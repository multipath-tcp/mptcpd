MPTCP Address Flags

Each MPTCP address flag is meant to be set as a bit in a mptcpd_flags_t variable as needed, e.g.:

mptcpd_flags_t flags =
MPTCPD_ADDR_FLAG_SUBFLOW | MPTCPD_ADDR_FLAG_BACKUP;

typedef void(* mptcpd_pm_get_addr_cb) (struct mptcpd_addr_info const *info, void *callback_data)

Type of function called when an address is available.

The mptcpd path manager will call a function of this type when the result of calling mptcpd_pm_get_addr() or mptcpd_pm_dump_addrs() is available.

Parameters
[in]infoNetwork address information. NULL on error.
[in,out]callback_dataData provided by the caller of mptcpd_pm_get_addr() or mptcpd_pm_dump_addrs().

typedef void(* mptcpd_pm_get_limits_cb) (struct mptcpd_limit const *limits, size_t len, void *callback_data)

Type of function called when MPTCP resource limit are available.

The mptcpd path manager will call a function of this type when the result of calling mptcpd_pm_get_limits() is available.

Parameters

[in]lenLength of the limits array. Zero on error.
