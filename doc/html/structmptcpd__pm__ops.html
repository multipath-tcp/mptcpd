Detailed Description

Path manager event tracking operations.

A set of functions to be called when changes in the path manager occur.

Note
These callbacks will only fire if the main event loop is running, e.g. through the ELL l_main_run() function. Users only need to be concerned about this if they are explicitly creating a struct mptcpd_pm instance instead of relying on mptcpd to do so, such what is done in some mptcpd unit tests.
