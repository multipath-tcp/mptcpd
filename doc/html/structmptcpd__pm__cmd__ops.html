Detailed Description

MPTCP path management generic netlink command functions.

The set of functions that implement MPTCP path management generic netlink command calls.

Advertise new network address to peers.

Parameters

[in]addrLocal IP address and port to be advertised through the MPTCP protocol ADD_ADDR option. The port is optional, and is ignored if it is zero.
[in]idMPTCP local address ID.
[in]flags
[in]indexNetwork interface index (optional for upstream Linux kernel).
[in]tokenMPTCP connection token.
